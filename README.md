## About GiftManager

GiftManager is a self-hosted web application built to solve the holiday dilemma of coordinating gifts among family and friends. It allows users to build personalized wishlists, securely claim gifts to avoid duplicates, and organize Secret Santa events.

Its core focus is on **preserving the surprise**: while family members can see what has been claimed by others, the recipient remains completely unaware. With robust support for multiple families, custom groups, and public sharing, it acts as a central hub for all your gifting occasions.

## Features

### Gift Ideas Management
- Add and edit gift ideas for yourself and others.
- Organize with **descriptions, links,** and prices.
- Automatically fetch Open Graph (OG) images from links or upload custom item pictures.
- Sort ideas however you like.
- Mark gifts as bought to **avoid duplicates.**
- Hide the purchaser globally or anonymously to maintain the element of surprise.
- Keep track of everything you’ve given.
- Built to keep **surprises secret**!
- Custom fields available for sizes, colors, or specific preferences.

### Nice Features
- **Secret Santa made simple:** Generate assignments with exclusion rules (so spouses don't draw each other).
- **Public Sharing:** Generate temporary, shareable links so external friends/family can view your list and claim gifts without needing an account.
- **Custom Avatars:** Upload your own profile picture or generate randomized SVG avatars.
- Emails notifications.
- Dark mode included.
- Manage multiple families separately.
- Guest accounts with highly restricted access to specific people or families.

### Global Management
- Easy-to-use **admin dashboard**.
- Built-in user management.
- Secure login with OIDC.
- Automated database cleanup for old purchased items.
